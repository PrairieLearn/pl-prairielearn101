A Hamiltonian cycle in a graph is a cycle that visits every vertex exactly once. Similarly, a Hamiltonian path in a graph is a path that visits every vertex exactly once.

Consider the following two decision problems:

  • HAM-CYCLE: Given an undirected graph $G$, decide whether or not it contains a Hamiltonian cycle.
  • HAM-PATH: Given an undirected graph $G$, decide whether or not it contains a Hamiltonian path.
