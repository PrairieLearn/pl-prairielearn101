 Wait until it receives an ACK or there is an ACK timeout. Send a jamming signal and cease transmission. Start the exponential backoff process. Wait for the transmission to end. Sense the medium to see if it is idle before transmitting. Sense the medium to be sure there is no collision. Transmit data. Give up.
