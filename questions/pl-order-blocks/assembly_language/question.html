Given two integer arrays of the same length, find the number of differences in the two arrays. A difference in the array is when at the same index value in the arrays, the integers values differ.

For example, [1,2,3] and [1,2,4] have 1 difference at index 2 since 3 is not equal to 4.

@paramter: r0 address to first int array

@paramter: r1 address to second int array

@paramter: r2 the length of the arrays (positive number)

@return: in r0, the number of different values in the array

You may assume that registers r3 - r6 have already been initialized to values of 0.

Arrange code to be efficient (Hint: this means at the beginning of the loop, a comparison to the end should be present.

.LOOP: cmp r3, r2 beq .ENDLOOP ldr r4, [r0]
ldr r5, [r1] cmp r4, r5 beq .MATCH add r6, r6, #1 .MATCH:
add r3, r3, #1
add r0, r0, #4
add r1, r1, #4 b .LOOP .ENDLOOP: move r0, r6
