Suppose that the position of a particle at time $t$ seconds is given by:

$$x(t) = {{params.x}}$$

where $x$ is measured in metres from the origin. Write a program in C named lab1.c that prompts the user for a time $t$ in seconds (we assume that $0\leq t \leq \pi$ but it is not necessary to check that the value entered by the user satisfies this condition). The program should then print the position of the particle in metres at the given time. The position must be printed to 3 decimal places.

You might find some of the following functions from the library math.h useful (all functions receive a double parameter and return a double value:

/* returns the square root of num */ sqrt(num) /* returns base to the power of exponent */ pow(base, exponent); /* returns sine of angle (angle measured in radians) */ sin(angle); /* returns cosine of angle (angle measured in radians) */ cos(angle);
