Now, let's come back to the idea of representing a matrix in C++: If we are representing a matrix with a vector of vector, each vector represents a row. Elements within represent different columns. Unlike matrices, vectors of vectors are not required to be "rectangular".
Exercise 1
Which statement is correct? When using a vector of vectors to represent a matrix of data, the outer vector represents the rows of the matrix, and the inner vectors represent the columns of the matrix. When using a vector of vectors to represent a matrix of data, the outer vector represents the columns of the matrix, and the inner vectors represent the rows of the matrix.

Exercise 2
You have the variable `data` that represents this vector of vectors: { 2 8 65 12 } { -4 0 32 -5 } { 1 -23 9 35 }
Which statement correctly prints the value of the element in the third "row" and second "column" of `data`? cout << data.at(2).at(1) << endl; cout << data.at(1).at(2) << endl; cout << data.at(3).at(2) << endl; cout << data.at(2).at(3) << endl;
