This image of `v2` is very helpful for understanding what is in `v2`: But C++ can't make this representation of `v2`. We need to figure out a different way to print out the contents of a vector of vectors. Something like this should work okay: { 5 2 23 8 } { 42 42 } { 2 3 4 5 11 13 }
This isn't the prettiest representation of the data in `v2`, but we can still see all the values so it's fine for our purposes here. The first "row" in this representation corresponds to the vector in `v2.at(0)` (or `v2[0]` if you prefer to use the `[]` method), the second "row" corresponds to the vector in `v2.at(1)` (or `v2[1]` ), and so on until all the data is printed. The built-in `<<` operator won't work to directly print out a vector of vectors in this format, but that's okay. We can **write a function to print out the values of a vector of vectors**, like we did when we needed to print out a struct or a vector of structs.
